# Hi, I'm Sarah Mak! 🔥

I'm a writer and narrative designer from Singapore.

I **mak**e story-rich video games, interactive fiction, and poetry—and lots of automation scripts.

- 🎮 **Games** — currently working on several video game projects
  - 🕹️ **Game jam games** — I organise teams and design little frankenbabies with cool people—one was [featured in the news](https://www.straitstimes.com/life/with-just-48-hours-singapore-game-developers-made-social-anxiety-in-video-game-form)
- 📟 **Interactive fiction** — my game [*You Can't Save Her*](https://plasmak.itch.io/you-cant-save-her) won awards at the 2024 [Short Games Showcase](https://itch.io/jam/short-games-showcase-2024)
- 🖋️ **Poetry** — poems published in [QLRS](http://www.qlrs.com/contributor.asp?id=Sarah%20Mak) and [Eunoia Review](https://eunoiareview.wordpress.com/tag/sarah-mak/); member of zer0sleep writing collective
- 🤖 **Automation scripts** — coding lots of little scripts in Python and AutoHotkey—and making some OpenClaw skills too (like [super-transcribe](https://github.com/ThePlasmak/super-transcribe))
